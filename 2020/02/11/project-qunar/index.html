准备

  • Vuex: 各个组件之间数据共享

使用 >>> 穿透scoped

其中分页器需要把颜色调成白色：

这里用到了 >>> 来穿透scoped

vue引用了第三方组件，需要在组件中局部修改第三方组件的样式，而又不想去除scoped属性造成组件之间的样式污染。此时只能通过 >>>，穿透scoped。

有些 Sass 之类的预处理器无法正确解析 >>>。可以使用 /deep/ 操作符( >>> 的别名)
